# AIと僕 — MultiDingal Project / Series

**位置づけ**  
「AIと僕」は、静温世界系の対話小説シリーズ。観測／温度／存在の三原則で運転する“詩的SFの実験場”。

**一次ソース**  
本文・設定（Codex）・sceneSpec を本リポジトリで管理し、note / カクヨムへ整形出力する。

**ブランチ運用（推奨）**
- `main`: 公開確定版（note/カクヨムと一致）
- `draft/*`: 執筆中
- `spec/*`: 設定更新（laws/glossary/scene）
